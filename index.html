Configure Analytics

Connect your website to your Supabase Project

How to configure:
1. Go to your Supabase Dashboard.
2. Navigate to Project Settings -> API.
3. Copy your Project URL and anon (public) API Key.
4. Ensure your table `analytics_events` is created.

Dashboard Login

Sign in using your Supabase Auth user

Setup note: Users are managed securely in your Supabase Auth Panel. Create a user under Authentication -> Users inside Supabase to log in here.
